## Tech Stack

- Next.js 14 - React 19 - Gsap - Tailwind CSS - MapLibre GL JS - React Map GL - Lenis

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
